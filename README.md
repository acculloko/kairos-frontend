# KairosFrontend

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 19.2.1.

## Installation guide
### Requirements:
NPM

### Clone Repository:
```
git clone https://github.com/acculloko/kairos-frontend.git
```

### Install dependencies:
```
npm install
```

### Run the application:
```
npm start
```
